# Edulance AI

Professional foundation for an AI-powered EdTech + freelancing SaaS platform.

## Project Overview

- **Frontend:** React, Vite, Tailwind CSS, React Router DOM, Axios.
- **Backend:** Node.js, Express.js, MongoDB Atlas, Mongoose.
- **Auth:** JWT + bcrypt.
- **Storage:** Cloudinary placeholders.
- **AI Integration:** Gemini API placeholder.

## Structure

- `frontend/` — React application scaffolding.
- `backend/` — Express API scaffolding.

## Setup

### Frontend
```bash
cd EdulanceAI/frontend
npm install
npm run dev
```

### Backend
```bash
cd EdulanceAI/backend
npm install
npm run dev
```

## Notes

- This sprint focuses on clean architecture, reusable components, and secure foundation.
- Advanced application flows and AI integrations are deferred to later sprints.
